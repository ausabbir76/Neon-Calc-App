# 🚀 Notes & Todos: A Multi-Themed Productivity App

Notes & Todos is a visually stunning, feature-rich productivity application built with Flutter. It combines the simplicity of note-taking with the utility of checklists, all wrapped in a highly customizable, theme-driven experience.

Whether you're into the high-contrast aesthetics of **Cyber** and **Neon**, or prefer the refined elegance of **Glass** and **Noir**, this app offers a unique visual environment tailored to your style.

---

## ✨ Key Features

- **📝 Dynamic Note-Taking**: Create, edit, and manage notes with a powerful, auto-saving editor.
- **✅ Checklist Management**: Build structured todo lists with reorderable tasks, completion tracking, and progress visualization.
- **🎨 Immersive Theming**: Choose from over 10+ premium themes, including:
    - **Dynamic Syncing**: Automatically switch between light and dark modes based on system settings.
    - **Visual Effects**: Enjoy glow effects, glassmorphism, and custom animations.
- **🔍 Advanced Search**: Instantly find notes or todos with a real-time, fuzzy-search engine.
- **♻️ Smart Management**:
    - **Archive**: Keep your workspace clean by archiving completed projects.
    - **Trash & Recovery**: Safely delete items with a dedicated trash bin and multi-step undo system.
    - **Pinning**: Keep your most important items at the top.
- **📊 Productivity Stats**: Visualize your progress and activity with built-in statistics.
- **📱 Responsive Design**: Fully optimized for various screen sizes with custom scaling support.
- **🔒 Local & Secure**: Your data stays on your device using high-performance SQLite storage.

## 🛠️ Technical Stack

- **Framework**: [Flutter](https://flutter.dev/)
- **Language**: [Dart](https://dart.dev/)
- **Database**: [SQLite](https://www.sqlite.org/) (via `sqflite`)
- **State Management**: `ChangeNotifier` & `ListenableBuilder`
- **Persistence**: `shared_preferences`
- **Icons**: Custom Theme-based Iconography
- **Architecture**: Repository Pattern with Controller-based State Orchestration

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (Latest Stable)
- Android Studio / VS Code with Flutter extension

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/notes_todos.git
   ```
2. Navigate to the project directory:
   ```bash
   cd notes_todos
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
